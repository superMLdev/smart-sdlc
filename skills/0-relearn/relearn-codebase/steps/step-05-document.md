## Overview

Generate each selected document using all findings from Steps 01–04. Each document is self-contained and should remain accurate as long as the code it describes doesn't change.

- {key feature 3}

| Layer | Technology |
|-------|-----------|
| {Runtime} | {Node 20 / Python 3.12 / Go 1.22} |
| {Framework} | {Express / FastAPI / Gin} |
| {Database} | {PostgreSQL via Prisma} |
| {Cache} | {Redis} |

## Prerequisites
- {Node.js ≥ 20 / Python ≥ 3.11 / Go ≥ 1.22}
- {Docker and Docker Compose}
- {PostgreSQL 14+}

## Quick Start

```bash
# Clone and install
git clone {repo-url}
cd {project-name}
{npm install | pip install -e . | go mod download}

# Configure
cp .env.example .env
# Edit .env with your values

# Start services (if docker-compose available)
docker compose up -d

# Run database migrations
{npm run migrate | alembic upgrade head | ...}

# Start development server
{npm run dev | uvicorn app.main:app --reload | go run cmd/main.go}
```

## Running Tests

```bash
{npm test | pytest | go test ./...}
```
