# smart-sdlc

Short-name alias for [`@supermldev/smart-sdlc`](https://www.npmjs.com/package/@supermldev/smart-sdlc).

Lets you use:
```bash
npx smart-sdlc init
npx smart-sdlc persona
npx smart-sdlc help
```

instead of the full scoped package name.

## Usage

```bash
npx smart-sdlc init
npx smart-sdlc persona
npx smart-sdlc help
npx smart-sdlc list
npx smart-sdlc meeting
npx smart-sdlc update
npx smart-sdlc clean
```

See [full documentation](https://github.com/superMLdev/smart-sdlc) for all commands and options.
